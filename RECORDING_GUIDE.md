# 录屏指导（观察者模式）

这版录屏的重点是：  
你**不能控制角色**，只能观察主角和 NPC 的自主行为，并向世界添加信息。

建议演示 5-7 分钟。

## 一、录屏前准备

1. 进入目录：`e:\world`
2. 安装依赖：`pip install requests`
3. 启动程序：`run.bat`
4. （可选）启动 Ollama：`ollama run llama3.2:3b`

## 二、推荐演示流程

### 1) 开场（20 秒）

- 口播建议：  
  “这是观察者模式的封闭文本世界，主角和 NPC 都是自主行动，我只能观察和改动世界信息。”

### 2) 展示主角自治（1-2 分钟）

连续输入多次：

```text
继续
继续
继续
```

观察输出中的：

- `[主角决策]`（行动类型和原因）
- `[主角行动]`（移动、等待、观察或对话）
- `[NPC回应]`（如果发生了对话）

口播建议：  
“这里不是我在命令角色，而是主角自己决定下一步做什么。”

### 3) 展示学习与永久记忆（1 分钟）

输入：

```text
查看记忆
```

检查：

- 记忆中有主角近期行为记录
- 持续存在“我是人类”永久记忆

### 4) 展示“添加世界但不控角色”（1-2 分钟）

输入：

```text
添加事实 今天广场钟楼比平时早响了十分钟
添加事物 广场 老陈书店门口新放了两盆薄荷
查看状态
继续
```

观察：

- 世界状态里出现你添加的事实/事物
- 主角后续回合会基于新环境继续自主行动

### 5) 展示 NPC 人格差异与非模板化对话（1-2 分钟）

再输入几次：

```text
继续
继续
继续
```

关注：

- `老陈` 与 `刘姐` 回复口吻不同
- 同一 NPC 多次对话不完全重复（LLM 动态生成）
- NPC 仍保持各自设定风格

### 6) 收尾（10 秒）

输入：

```text
帮助
退出
```

## 三、成功判定清单

- [ ] 画面中出现主角自主决策与行动日志
- [ ] 展示了“我是人类”永久记忆
- [ ] 演示了观察者可添加事实/事物但不能下达角色行为指令
- [ ] 演示了老陈和刘姐的风格差异且回复非固定模板

## 四、常见问题

- 看起来回复比较保守：通常是 Ollama 未连接，已走回退逻辑（功能正常）
- 主角暂时没触发对话：多输入几次 `继续`，让世界多跑几回合
- 想加剧情线索：用 `添加事实 ...` 或 `添加事物 <地点> <内容>`
